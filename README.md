## ¿Qué es DemoSalonCejas?

DemoSalonCejas es una demo funcional de plataforma de reservas para salones de cejas y estética, desarrollada por Xyon Platforms. Permite a los clientes consultar servicios y reservar citas online, y a los administradores gestionar la agenda del negocio.

## ✨ Funcionalidades

- **Catálogo de servicios** de estética
- **Reservas online** para clientes
- **Panel de administración** de citas y agenda
- **Diseño responsive** para móvil y escritorio

## 🛠️ Stack técnico

| Capa | Tecnología |
|------|----------|
| Frontend | React + TypeScript + Vite |
| Backend | Node.js + TypeScript |
| Despliegue | Docker + Coolify |

## 🚀 Instalación

```bash
git clone https://github.com/mkandreum/demosaloncejas.git
cd demosaloncejas
cp .env.example .env
docker-compose up -d
```

### Desarrollo local

```bash
npm install
npm run dev
```
